celt_codec module usage guide

Brief:          This module is the audio codec implementation.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/audio/module.mk" in your GCC project Makefile.
                    MTK_AUDIO_CELT_ENABLE must be set to "y".
