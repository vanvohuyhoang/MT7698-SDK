ogg_celt module usage guide

Brief:          This module is the ogg audio codec implementation.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/audio/module.mk" and include the module with "include $(SOURCE_DIR)/middleware/MTK/ogg_celt/module.mk"
                in your GCC project Makefile. MTK_AUDIO_CELT_ENABLE must be set to "y".
Dependency:     None
Notice:         None
Relative doc:   None
